104. 二叉树的最大深度

给定一个二叉树，找出其最大深度。

二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。

说明: 叶子节点是指没有子节点的节点。

示例：给定二叉树 [3,9,20,null,null,15,7]，

返回它的最大深度 3 。

  • 第一种做法，深度 = (左子树最大深度，右子树最大深度) + 1
  • 第二种做法，维护一个全局变量，每次递归结束时，更新全局变量的值
