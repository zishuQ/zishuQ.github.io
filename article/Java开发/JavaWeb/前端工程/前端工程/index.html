Ajax

Ajax：Asynchronous JavaScript And XML，异步的JavaScript和XML

  • 作用：
    • 数据交换：通过Ajax可以给服务器发送请求，并获取服务器响应的数据
    • 异步交互：可以在不重新加载整个页面的情况下，与服务器交换数据并更新部分网页的技术

原生Ajax

  1. 创建XMLHttpRequest对象：用于和服务器交换数据

  2. 向服务器发送请求

  3. 获取服务器响应数据

Axios

Axios对原生的Ajax进行了封装，简化书写，快速开发

步骤

  1. 引入Axios的js文件

  2. 使用Axios发送请求，并获取响应结果

  • 请求方式别名：

    • axios.get(url [, config])
    • axios.delete(url [, config])
    • axios.delete(url [, data[, config]])
    • axios.put(url [, data[, config]])

  • 发送GET请求

  • 发送POST请求

前端工程化

环境准备

vue-cli

  • 介绍：Vue-cli是Vue官方提供的一个脚手架，用于快速生成一个Vue的项目模板
  • Vue-cli提供了如下功能：
    • 统一的目录结构
    • 本地调试
    • 热部署
    • 单元测试
    • 集成打包上线
  • 依赖环境：Nodejs

现在官方推荐使用Vite

vue项目

项目目录结构

  1. 创建

    • 命令行

    • 图形化界面

  2. 启动

    切换端口： 修改vue.config.js文件

  3. 开发流程

    • Vue的组件文件以 .vue结尾，每个组件由三个部分组成：<template>、<script>、<style>

      • template：模板部分，由它生成HTML代码
      • script：控制模板的数据来源和行为
      • style：css样式部分

    • 示例

Vue组件库Element

  • Element是饿了么团队研发的，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

  • 组件：组成网页的部件，例如 超链接、按钮、图片、表格、表单、分页条等等

快速入门

  • 安装ElementUI组件库

  • 引入ElementUI组件库到 main.js

  • 访问官网，复制组件代码，调整

  • 在App.vue中使用<element-view>标签进行引入

前端路由

前端路由：URL中hash(#号)与组件之间的对应关系

  • Vue路由是Vue的官方路由
  • 组成：
    • VueRouter：路由器类，根据路由请求在路由视图中动态渲染选中的组件
    • <router-link>：请求链接组件，浏览器会解析成<a>
    • <router-view>：动态视图组件，用来渲染展示与路由路径对应的组件

  • 安装

  • 定义路由 router/index.js

打包部署

使用命令 npm run build进行打包操作，会生成一个 dist文件夹，把这个文件夹上传到服务器（前端服务器一般使用nginx）
