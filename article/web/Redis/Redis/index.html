Redis简介

Redis是Remote Dictionary Server的缩写，是一个开源的基于内存的数据存储系统

五种基本数据类型：

  1. 字符串 String
  2. 列表 List
  3. 集合 Set
  4. 有序集合 SortedSet
  5. 哈希 Hash

五种高级数据类型：

  1. 消息队列 Stream
  2. 地理空间 Geospatial
  3. HyperLogLog
  4. 位图 Bitmap
  5. 位域 Bitfield

字符串 String

  • 命令
    1. SET：设置键值对
    2. GET：获取键值对的值
    3. DEL：删除键值对
    4. EXISTS：判断键值对是否存在
    5. KEYS：列出匹配的键值对
    6. FLUSHALL：删除所有键值对
    7. redis-cli —raw：使存储的数据能够正常显示（中文）
    8. TTL：查看键值对过期时间
    9. EXPIRE：设置过期时间
    10. SETEX：创建键值对的时候加入过期时间
    11. SETNX：在 key 不存在的时候创建键值对
  • 对键值对的操作 set get del exists
  • 匹配键值对 keys 清空键值对 flushall
  • 查看中文键值对
  • 设置键值对过期时间 ttl expire setex setnx

列表 List

  • 命令
    1. LPUSH：从左侧加入元素
    2. LPOP：从左侧弹出元素
    3. RPUSH：从右侧加入元素
    4. RPOP：从右侧弹出元素
    5. LRANGE：遍历列表 start 到 stop 的元素
    6. LLEN：显示列表长度
    7. LTREM：删除除了 start 到 stop 之外的元素
  • 操作示例

集合 Set

  • 命令
    1. SADD：加入元素
    2. SMEMBERS：查看集合中的元素
    3. SISMEMBER：判断元素是否在集合中
    4. SREM：删除集合中的元素
    5. SINTER：取集合的交集
    6. SUNION：取集合的并集
    7. SDIFF：取集合的补集
